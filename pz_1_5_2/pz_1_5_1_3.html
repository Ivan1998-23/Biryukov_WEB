HTML тег < HEADER >

Назва типу < header >

Тег < header > призначений для того, щоб позначити розділ HTML-сторінки як заголовок (контейнер для вступного вмісту) або набір навігаційних посилань.

Елемент < header > зазвичай містить:

  1. один або кілька заголовків елементів (< h1> - < h6 >)
  2. логотип або значок
  3. інформацію про авторство

У HTML-документі може міститися одночасно кілька елементів < header > і вони можуть розташовуватися в будь-якій частині сторінки.

Синтаксис

< header > Заголовок < /header >

data-* Визначає групу атрибутів (атрибути призначених для користувача даних), які дозволяють обмінюватися службовою інформацією між HTML і його DOM поданням. Будь-яке значення, яке описує намір користувача
dir Визначає напрямок тексту в елементі (зліва направо або справа наліво) ltr - left to right (зліва направо) rtl - right to left (справа наліво) auto - напрямок тексту визначається браузером
draggable Вказує, чи можна перетягувати елемент використовуючи Drag and Drop API It is used to define if an element is draggable or not. true - перетягування елемента дозволено false - перетягування елемента заборонено
dropzone Визначає, що робити з перетягувати користувачем даними copy (за замовчуванням) - скопіювати стерпний елемент до приймаючого елемент. link - створити в елементі посилання на першоджерело, звідки він був переміщений. move - перенести елемент до приймаючого елемент, видаливши його з попереднього
hidden Вказує, що вміст елемента більш не актуальне і має бути приховане від перегляду. Порожній рядок або ім'я атрибута (hidden). Атрибут може бути вказаний без значення.
id Встановлює унікальний ідентифікатор для елемента, до якого можна звернутися за створення посилань на нього, виконанні скриптів або застосуванні стилів. Тільки англійські букви (верхнього і нижнього регістра), цифри, а також символи «-» (дефіс) і «_» (підкреслення)

spellcheck Вказує браузеру перевіряти чи текст на наявність орфографічних помилок. true - елемент повинен перевірятися false -Елемент не повинен перевірятися

translate Вказує, чи слід переводити текстовий вміст і значення атрибутів елемента. yes - перевести no - не перекладати
